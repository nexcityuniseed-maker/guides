— 01 / OVERVIEW

What is Claude Design? そもそもClaude Designって？

Claude Designは、2026年4月17日にAnthropicがリリースしたばかりの新しいAIデザインツールです。

最大の特徴は「話しかけるだけ」でプロトタイプ・スライド・チラシ・LP・SNS素材などのビジュアルが作れること。Figmaのような専門ツールの操作を覚える必要はありません。

裏側ではClaude Opus 4.7という最新の高性能AIが動いていて、デザインのセンスや解像度がこれまでより大幅に向上しています。

できあがったデザインはPDF・PowerPoint・URL・Canvaに書き出し可能。Canvaに送ればチームで編集もできます。

— こんな人にぴったり

"デザインソフトは使えないけど、
イメージを形にしたい"

  • Canvaでチラシを作るのに毎回時間がかかる店長
  • HPB用のキャンペーン画像をサクッと作りたいスタッフ
  • 本部に送る企画書・提案資料を作りたい人
  • 新メニューのプロモ素材をすぐに見たい人
  • SNS投稿のテンプレデザインを作りたい人

— 03 / HOW TO USE

How to use it. 使い方の基本ステップ

01

Claude.aiにログインして、サイドバーの「Design」アイコンをクリック

パソコンのブラウザで claude.ai を開いてログイン。左側のサイドバーにパレット型のアイコンがあるので、それをクリックします。これがClaude Designの入り口。

NOTE 機能は順次ロールアウト中のため、まだ表示されない場合があります。その場合は数日待つか、Anthropicのアカウント設定で機能をONにしてみてください。
02

「Let's prototype」のチャット欄に作りたいものを話しかける

画面の真ん中にチャット欄が出てくるので、普段Claudeに話しかけるのと同じ感覚で「何を作りたいか」を伝えるだけ。難しい指示は要りません。

eye mor.の母の日キャンペーン用チラシを作って。
  ・対象：30〜50代の女性
  ・内容：マツエク20%OFFクーポン
  ・期間：5/1〜5/12
  ・トーン：上品でゴールドっぽい高級感
03

参考画像・PDF・資料があればアップロード

手書きのラフスケッチ、過去のチラシ、ブランドガイドのPDF、商品の写真など、イメージのもとになるものを一緒にアップするとクオリティが一気に上がります。

DOCX・PPTX・XLSXファイルもそのまま読み込めるので、既存の企画書からスライドを作ってもらうこともできます。

04

できあがったデザインを「Tweaks」で微調整

AIが最初のデザインを出してくれたら、「Tweaks」ボタンで細かい修正ができます。気になる部分をクリックして「ここの色をピンクに」「フォントをもう少し細く」など、会話形式で直していけます。

TIP 一発でOKを目指さず、3〜5回会話で調整していくのがコツ。「もう少し○○に」「✕✕の要素を入れて」と気軽に伝えましょう。
05

完成したらエクスポート（PDF / PPTX / Canva / URL）

納得のいくデザインになったら、用途に合わせて書き出し。Canvaに送ればそのまま編集・共有・印刷発注ができますし、PDF・PPTXでもダウンロード可能です。

URL書き出しなら、LINEやメールでスタッフに即共有できます。

→ 基本はこっちを選ぶ（最初から選択されている青枠）。スタッフには「青い枠の右側のまま」と伝えればOK。

A〜H：作業画面の全項目を解説

A
Chat pane
左側：チャットパネル 操作の中心

AIに日本語で指示を出す場所。「○○を作って」「ここを直して」「もっと明るく」など、思いついたことをそのまま書いてOK。普段のChatGPTやClaudeと同じ感覚。

B
User message
あなたの発言（黒い吹き出し）

自分が送ったメッセージは黒地に白文字で右寄せ表示。会話の履歴がスクロールで遡れます。

C
AI response
AIの返事（白い吹き出し）

Claudeからの返答は白地に黒文字で左寄せ。指示が曖昧な時は「◯◯はどうしますか？」と質問してくれます。その質問に答えるだけでクオリティが上がるので無視しないこと。

D
📷 / 📄 / ↑ Attach & Send
添付・送信ボタン よく使う

入力欄の右側にある3つのアイコン。
・📷：参考画像をアップロード（過去のチラシ・好きなデザインなど）
・📄：PDF・Word・Excelなどファイル添付
・↑：送信ボタン（Enterキーでも送れる）

E
Canvas
右側：キャンバス 作品が出る場所

AIが作ったデザインがリアルタイムで表示されるエリア。スライド資料なら複数枚が縦に並びます。マウスでクリック・選択して直接編集もできます。

F
Tweaks / Comment / Edit / Draw
4つの編集ツール 次のセクションで詳説

キャンバス上部の細かい修正用ツールバー。
・Tweaks：つまみで調整
・Comment：場所指定でコメント
・Edit：その場で書き換え
・Draw：手描き指示
→ 詳しくは「06 / EDIT TOOLS」で。

G
Export ↗
右上：書き出しボタン 完成したら押す

デザインが完成したら必ず最後に押すボタン。PDF / PPTX / Canva / HTML / ZIP / Claude Codeの6種類から選んで保存。
→ 詳しくは「07 / EXPORT」で。

H
Design output

🎯 こんな時どうする？場面別ガイド

💡

イメージを伝えたい

📷ボタンで参考画像をアップ。「こんな感じで」と添えると、雰囲気が伝わりやすい。スマホで撮った写真でもOK。

✏️

文字を直したい

キャンバス上のEditボタンを押してから、文字をクリック→打ち直し。AIに頼まず一瞬で書き換えできる。

📍

場所を指定して直したい

Commentボタンを押してから、直したい部分をクリックしてピン留め。「ここの色を変えて」と書くだけでOK。

🔄

違う案も見たい

チャットで「2〜3パターン他の案も見せて」と頼むだけ。複数バリエーションを比較してから決められる。

🌐

スマホ表示も確認したい

「スマホでも見やすいレイアウトにして」と伝える。WEBページや LP の場合は最初から指定しておくとベター。

🆘

変な方向に行ってしまった

「今の状態を保存して、別のアプローチを試して」と伝えれば、過去案も残しつつ別案を試せる。

⚠ よくあるミス

・1回で完成させようとしない：3〜5回のやり取りで仕上げるのが基本。最初から完璧な指示を出そうとしなくてOK。
・キャンバスをスクロールするのを忘れる：スライド資料は縦に複数枚並ぶ。下までスクロールして全体を確認。
・Exportを忘れて閉じる：作業内容は自動保存されるが、書き出さないと使えない。最後にExportボタン必須。

— 06 / EDIT TOOLS

4つの編集ツール。 細かい修正はこれで使い分け

キャンバスができあがったら、上部に並ぶ4つのツールを使って微調整します。それぞれ用途が違うので、シーン別に覚えておくと便利。

A Tweaks

つまみで調整

トゥイークス

スライダーやつまみのような調整パネルがAIにより動的に作られます。「文字を大きく」「余白を広く」など、感覚的に動かせる微調整に最適。

こんなとき： 全体のバランス調整、サイズ感の微調整
B Comment

ピンを刺してコメント

コメント

画面上の特定要素をクリックしてピンを刺し、メモを残す方式。「このボタンの色を変えて」「ここの余白広く」など、場所を指定したい修正に強い。Googleドキュメントのコメント機能のイメージ。

こんなとき： 「ここを直して」と場所を指定したい時
C Edit

その場で書き換え

エディット

テキストをクリックしてそのまま打ち直すだけ。AIに頼まず即書き換わるので、誤字修正や文言差し替えが超スピーディ。一番早い修正方法。

こんなとき： 誤字、文言の差し替え、価格や日付の修正
D Draw

キャンバスに直接描く

ドロー

画面の上にマウスやペンで矢印・丸・手書き文字を直接描き込み、AIがそれを指示として読みます。「ここに何か入れて」「この向きに動かして」など、言葉で説明しづらい修正に強い。

こんなとき： 言葉で説明しにくいレイアウト変更
使い分けのコツ

誤字直し → Edit（一番早い）／ 場所指定の修正 → Comment（精度が高い）／ 全体の調整 → Tweaks（バランス調整）／ 言葉に出来ない時 → Draw（最後の手段）／ 大きな構成変更 → 左のチャットに直接書く

— 09 / TIPS

上手に使うコツ。 伝え方のポイント5つ

✓

誰に・何を伝えたいかを書く

「ターゲット」「目的」「使う場所」を最初に伝えると、ピントの合ったデザインが返ってきます。

✓

トーン・雰囲気を言葉で表現

「上品」「ポップ」「和モダン」「シンプル」など、ムードを言葉で。色のイメージも添えると◎。

✓

参考画像を必ずアップ

過去のチラシ、好きなデザインのスクショなど、ビジュアル参考があるだけで仕上がりが段違い。

✓

サイズ・媒体を指定する

「A4チラシ」「Instagram正方形」「HPBバナー(300×250)」など、出口のサイズを明示。

✓

一発OKを狙わない

3〜5回の会話で仕上げる前提で。「ここを直して」を繰り返す方が、結果的に早くて良いものに。

✓

困ったらCanvaに送る

細かい文字差し替えや写真の入れ替えはCanva側で。AIで土台、人の手で仕上げが最強コンビ。

— IMPORTANT NOTICE

使う前に知っておいてほしいこと。

  • 現在は「リサーチプレビュー」段階。バグや動作の不安定さが残っています。
  • 利用回数には独自の上限があり、通常チャットとは別枠で消費されます。
  • お客様の個人情報・カルテ情報・売上数値などはアップしないでください。
  • 生成されたデザインは必ず人の目で最終確認。誤字や不適切表現がないかチェック必須。
  • ブランドロゴや店舗写真は公式素材のみを使用すること。AI生成素材を勝手に対外発信しない。
  • 使い方で迷ったら、本部または馬場まで気軽に相談してください。
